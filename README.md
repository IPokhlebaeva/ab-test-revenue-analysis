# A/B-тест: анализ влияния изменения на выручку

## Бизнес-задача

Оценить, приводит ли тестируемое изменение на сайте к **статистически значимому росту выручки на пользователя**.  
На основе результатов — дать рекомендацию: **раскатывать изменение или нет**.

## Структура проекта
AB.ipynb # Jupyter Notebook с полным анализом <br>
README.md # Описание проекта  <br>
requirements.txt # Зависимости для воспроизведения <br>
transactions.csv # Транзакции пользователей <br>
users.csv # Разбивка по группам (control/treatment) <br>
